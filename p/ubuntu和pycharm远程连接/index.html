Ubuntu和Pycharm远程连接

2025-03-03
阅读时长: 2 分钟

一、Ubuntu配置

在使用 PyCharm 连接 Ubuntu之前，需要保证 Ubuntu 虚拟机的 SSH 服务处于开启状态。你可以按照以下步骤操作：

  1. 开启 SSH 服务：sudo apt-get install openssh-server
  2. 查看服务状态：输入 sudo systemctl status ssh 来查看 SSH 服务是否已成功开启。若显示 active (running)，就表明服务已正常运行。如果显示dead则没有开启ssh可以尝试sudo systemctl start ssh
  3. 查看 IP 地址：输入 ip addr 或者 ifconfig 命令查看 Ubuntu 虚拟机的 IP 地址

二、实现Ubuntu系统和pycharm远程连接 2021.3.1专业版pycharm

（1）在pycharm软件中创建连接记录 菜单栏Tools > Deployment > Configuration > + > SFTP > Ubuntu的ip地址 > OK （2）配置连接记录，如图1-16所示，表示连接成功。 ··· > + > 配置ip、用户名、密码 > 测试连接

（3）配置部署关联目录 /home/Ubuntu/code local path ： 本地目录 > Deployment path： /home/Ubuntu/code

（4）配置项目解释器 导航栏File > setting > Project code > + Add Content > + > SSH Interpreter > 选择Existing serber configureation > ··· > 选择创建的root@地址 > Move
Interpreter：/home/Ubuntu/.virtualenvs/python_auto/bin/python3 设置 Sync folder为 <Project root>-/home/Ubuntu/code>OK>OK

提示：一定要确认映射文件目录是否有错，否则解释器失效。 （5）同步pycharm代码到Ubuntu系统，即完成准备工作。
